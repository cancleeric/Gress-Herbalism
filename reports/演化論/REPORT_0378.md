## 完成內容摘要

### 問題

前後端 Socket.io 心跳超時參數不一致：
- 後端 `pingTimeout: 30000`（30 秒）
- 前端 `pingTimeout: 20000`（20 秒）

當網路延遲介於 20-30 秒時，前端會誤判為斷線。

### 解決方案

將前端 `pingTimeout` 從 20000 改為 30000，與後端一致。

---

## 修改檔案

```
frontend/src/services/socketService.js    # 更新 pingTimeout 為 30000
```

---

## 修改內容

```javascript
// 修改前
pingTimeout: 20000,               // 心跳超時 20 秒

// 修改後
pingTimeout: 30000,               // 心跳超時 30 秒（與後端一致）
```

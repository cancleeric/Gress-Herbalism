## 測試執行方式

```bash

npm run cypress:run -- --spec "tests/e2e/edge/**/*.spec.js"

# 開啟 UI 模式
npm run cypress:open
```

## 測試模擬方式

使用 `localStorage` 模擬各種邊界情況：

```javascript
// 模擬牌庫為空
win.localStorage.setItem('mockDeckEmpty', 'true');

// 模擬斷線
cy.goOffline();

// 模擬慢速網路
cy.intercept('**/*', (req) => {
  req.on('response', (res) => res.setDelay(500));
});
```

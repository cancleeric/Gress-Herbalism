- **描述**：好友組件完全沒有測試覆蓋
- **原因**：需要複雜的 Socket 和 API mock
- **建議**：建立完整的好友功能測試，包括：
  - 好友列表顯示

  - 在線狀態顯示

### 問題 2：Lobby.js 覆蓋率偏低 (66.44%)
- **描述**：大廳組件有較多未測試的分支
- **原因**：複雜的 Socket 互動和房間管理邏輯
- **建議**：補充以下測試：
  - 創建房間流程
  - 加入房間流程
  - 密碼房間處理
  - 房間列表更新

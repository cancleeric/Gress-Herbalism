## 完成內容摘要

建立遊戲房間（GameRoom）組件的基礎結構。

### 實作內容

1. **`frontend/src/components/GameRoom/GameRoom.js`**
   - 函數式組件，使用 React Hooks
   - 從 URL 參數取得遊戲 ID (`useParams`)
   - 連接 Redux Store 取得遊戲狀態

2. **UI 佈局**
   - 頂部區域：遊戲標題、房間ID、遊戲階段、離開按鈕
   - 左側：玩家列表（顯示名稱、房主標記、當前回合標記）
   - 中央：遊戲桌面（等待/進行中/結束狀態）
   - 右側：遊戲歷史記錄
   - 底部：自己的手牌、問牌/猜牌按鈕

3. **遊戲階段管理**
   - 等待階段：顯示等待訊息、開始遊戲按鈕
   - 進行中：顯示當前回合玩家
   - 結束：顯示獲勝者、蓋牌揭曉

4. **Redux 連接**
   - `useSelector` 取得遊戲狀態
   - `useDispatch` 發送 actions
   - 離開房間時重置狀態

5. **`frontend/src/components/GameRoom/GameRoom.css`**
   - CSS Grid 三欄佈局
   - 響應式設計（平板/手機）
   - 卡牌漸層樣式
   - 當前回合玩家高亮

6. **更新 App.js**
   - 匯入並使用 GameRoom 組件

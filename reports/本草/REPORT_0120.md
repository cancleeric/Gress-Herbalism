## 五、整合測試指南

### 5.1 測試環境準備

1. 啟動後端服務：
   ```bash
   cd backend
   npm run dev
   ```

2. 啟動前端服務：
   ```bash
   cd frontend
   npm start
   ```

3. 準備多個瀏覽器視窗或無痕視窗

### 5.2 主要測試場景

工單 0120 定義了 14 個測試案例，分為以下類別：

1. **等待階段測試**（3 個案例）
   - WA-01：等待階段單人重整
   - WA-02：等待階段多人重整
   - WA-03：等待階段房主超時

2. **遊戲中測試**（3 個案例）
   - GP-01：遊戲中重整
   - GP-02：遊戲中網路中斷
   - GP-03：遊戲中長時間斷線

3. **LocalStorage 測試**（2 個案例）
   - LS-01：5 分鐘後重連
   - LS-02：模擬超過 2 小時

4. **Redux Persist 測試**（2 個案例）
   - RP-01：狀態恢復
   - RP-02：主動離開時清除

5. **UI 提示測試**（1 個案例）
   - UI-01：連線狀態顯示

6. **效能測試**（1 個案例）
   - PF-01：快速連續重整

7. **邊界情況測試**（2 個案例）
   - EC-01：重連時房間已被刪除
   - EC-02：重連時遊戲已結束

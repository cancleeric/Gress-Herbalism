## 建議實作方式

### 1. 安裝依賴

```bash
cd frontend
npm install --save-dev @playwright/test
npx playwright install
```

### 2. 配置檔

建立 `frontend/playwright.config.js`：
- 設定測試目錄
- 設定基礎 URL
- 設定瀏覽器配置
- 設定失敗截圖和影片

### 3. 測試案例建議

**基礎流程測試：**
- 訪客登入流程
- 建立房間流程
- 加入房間流程
- 完整遊戲流程

**多人遊戲測試：**
- 3-4 人遊戲模擬
- 問牌互動測試
- 猜牌流程測試
- 計分驗證測試

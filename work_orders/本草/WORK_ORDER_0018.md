# 工作單 0018

**日期：** 2026-01-23

**工作單標題：** 建立玩家手牌組件 - 手牌顯示

**工單主旨：** UI 組件 - 建立玩家手牌（PlayerHand）組件顯示玩家自己的手牌

**內容：**

## 工作內容

1. **建立 `frontend/src/components/PlayerHand/PlayerHand.js` 檔案**

2. **建立組件基礎結構**
   - 建立函數式組件
   - 接收玩家手牌作為 props（或從 Redux 取得當前玩家的手牌）

3. **實作手牌顯示**
   - 顯示玩家自己的所有手牌
   - 每張牌顯示其顏色（使用顏色標識或卡片樣式）
   - 手牌以水平排列顯示
   - 每張牌應該有清晰的視覺區分

4. **實作手牌樣式**
   - 不同顏色的牌使用不同的顏色標識
   - 可以使用卡片樣式或簡單的顏色方塊
   - 設定手牌的間距和大小

5. **實作手牌互動**（為未來擴展預留）
   - 手牌可以點選（為未來選擇手牌功能預留）
   - 手牌有 hover 效果

6. **連接 Redux Store**
   - 使用 `useSelector` 取得當前玩家的手牌
   - 當手牌變化時自動更新顯示

7. **建立樣式檔案**
   - 建立 `frontend/src/components/PlayerHand/PlayerHand.css`
   - 設定手牌的佈局樣式
   - 設定不同顏色牌的樣式
   - 設定手牌的動畫效果（可選）

8. **匯出組件**
   - 確保組件可以正確匯出和匯入

9. **使用 JSDoc 註解**
   - 為組件添加 JSDoc 註解

## 驗收標準

- [ ] `PlayerHand.js` 組件已建立
- [ ] 可以正確顯示玩家自己的手牌
- [ ] 每張牌正確顯示其顏色
- [ ] 手牌佈局美觀
- [ ] Redux 連接正確
- [ ] 手牌變化時自動更新

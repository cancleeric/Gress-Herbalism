# 工作單 0019

**日期：** 2026-01-23

**工作單標題：** 建立問牌介面組件 - 基礎結構

**工單主旨：** UI 組件 - 建立問牌介面（QuestionCard）組件的基礎結構

**內容：**

## 工作內容

1. **建立 `frontend/src/components/QuestionCard/QuestionCard.js` 檔案**

2. **建立組件基礎結構**
   - 建立函數式組件
   - 接收必要的 props（onSubmit, onCancel 等）
   - 使用 Modal 或 Dialog 形式顯示（或內嵌在遊戲介面中）

3. **實作顏色選擇器**
   - 顯示四個顏色選項：紅色、黃色、綠色、藍色
   - 允許選擇兩個不同的顏色
   - 已選擇的顏色要有視覺反饋
   - 驗證必須選擇兩個不同顏色

4. **實作目標玩家選擇器**
   - 顯示其他玩家列表（排除自己）
   - 允許選擇一個目標玩家
   - 已選擇的玩家要有視覺反饋

5. **實作要牌方式選擇器**
   - 顯示三種要牌方式選項：
     1. 兩個顏色各一張
     2. 其中一種顏色全部
     3. 給其中一種顏色一張，要另一種顏色全部
   - 允許選擇一種要牌方式
   - 已選擇的方式要有視覺反饋

6. **實作確認和取消按鈕**
   - 確認按鈕：提交問牌動作
   - 取消按鈕：關閉問牌介面

7. **建立樣式檔案**
   - 建立 `frontend/src/components/QuestionCard/QuestionCard.css`
   - 設定問牌介面的佈局樣式
   - 設定選擇器的樣式

8. **匯出組件**
   - 確保組件可以正確匯出和匯入

9. **使用 JSDoc 註解**
   - 為組件添加 JSDoc 註解

**工作單標題：** 斷線重連機制整合測試

## 一、測試目標

驗證工單 0115-0119 的所有修改是否正確運作，確保：

1. 等待階段斷線後能重連
2. 遊戲中斷線後能重連
3. 頁面重整後能自動重連
4. UI 正確顯示連線狀態

## 二、測試環境準備

### 2.1 前置條件

- [ ] 後端服務啟動 (`npm run dev`)
- [ ] 前端服務啟動 (`npm start`)
- [ ] 至少準備 2 個瀏覽器（或無痕視窗）模擬多玩家

### 2.2 測試工具

# 工作單 0075

**日期：** 2026-01-25

**工作單標題：** 顏色牌選擇記錄與限制

**工單主旨：** 功能開發 - 公開標記玩家選擇的顏色牌，並限制不能連續選同一張

**依賴工單：** 0073, 0074

## 功能概述

1. 每個玩家選擇顏色牌時，在該牌上標記玩家標識（公開資訊）
2. 玩家不能選擇自己上回合選過的顏色牌

## 功能需求

### 1. 公開標記機制

當玩家選擇顏色牌後，在該牌上顯示玩家標記：

```
┌─────────────────────────────────────────────────────────┐
│                    六張顏色組合牌                         │
│                                                         │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐                │
│   │  紅綠   │  │  綠藍   │  │  綠黃   │                │
│   │         │  │         │  │         │                │
│   │ [小明]  │  │         │  │ [小華]  │                │
│   └─────────┘  └─────────┘  └─────────┘                │
│                                                         │
│   ┌─────────┐  ┌─────────┐  ┌─────────┐                │
│   │  紅藍   │  │  黃紅   │  │  黃藍   │                │
│   │         │  │         │  │         │                │
│   │         │  │ [小王]  │  │         │                │
│   └─────────┘  └─────────┘  └─────────┘                │
│                                                         │
│   標記說明：顯示每位玩家上回合選擇的牌                     │
└─────────────────────────────────────────────────────────┘
```

### 2. 標記樣式

| 元素 | 說明 |
|------|------|
| 玩家標記 | 顯示玩家暱稱或頭像 |
| 標記位置 | 卡牌底部或角落 |
| 標記顏色 | 對應玩家的識別色 |

**標記範例：**

```
┌─────────────┐
│ 🔴       🟢 │
│             │
│   [圖案]    │
│             │
│  ┌───────┐  │
│  │ 小明  │  │  ← 玩家標記
│  └───────┘  │
└─────────────┘
```

### 3. 選擇限制規則

| 規則 | 說明 |
|------|------|
| 限制對象 | 僅限制自己，不限制其他玩家 |
| 限制範圍 | 不能選擇自己**上一回合**選過的牌 |
| 新局重置 | 每局開始時清除所有標記 |
| 第一回合 | 無限制，可選任意牌 |

### 4. 禁用提示

當玩家嘗試選擇禁用的牌時：

```
┌─────────────────────────────────────┐
│  ⚠️ 無法選擇                        │
│                                     │
│  你上回合已經選過這張牌，            │
│  請選擇其他顏色組合。                │
│                                     │
└─────────────────────────────────────┘
```

### 5. 視覺呈現

**自己禁用的牌：**
```
┌─────────────┐
│ 🔴       🟢 │
│   ░░░░░░░   │  ← 灰色遮罩
│   ░圖案░░   │
│   ░░░░░░░   │
│  ┌───────┐  │
│  │ 你    │  │  ← 標示「你」
│  └───────┘  │
│     🚫      │  ← 禁止圖示
└─────────────┘
```

**其他玩家標記的牌（可選）：**
```
┌─────────────┐
│ 🔴       🟢 │
│             │
│   [圖案]    │  ← 正常顯示
│             │
│  ┌───────┐  │
│  │ 小華  │  │  ← 其他玩家標記
│  └───────┘  │
└─────────────┘
```

## 資料結構

### 遊戲狀態

```javascript
const gameState = {
  // 每位玩家上回合選擇的顏色牌
  lastColorCardSelection: {
    'player1': 'red-green',   // 小明上回合選了紅綠
    'player2': 'green-yellow', // 小華上回合選了綠黃
    'player3': 'yellow-red',   // 小王上回合選了黃紅
    'player4': null,           // 小李還沒選過（第一回合）
  },
};
```

### 事件廣播

```javascript
// 當玩家選擇顏色牌後，廣播給所有玩家
socket.broadcast('colorCardMarked', {
  playerId: 'player1',
  playerName: '小明',
  cardId: 'red-green',
});
```

## 流程說明

```
回合開始：
1. 檢查當前玩家上回合選過的牌
2. 將該牌設為禁用狀態
3. 玩家選擇顏色牌
4. 更新 lastColorCardSelection
5. 廣播標記更新給所有玩家
6. 繼續問牌流程

新局開始：
1. 清除所有 lastColorCardSelection
2. 移除所有卡牌標記
```

# 工作單 0076

**日期：** 2026-01-25

**工作單標題：** 問牌後未顯示預測選項

**工單主旨：** BUG 修復 - 問牌完成後應顯示預測選項但未出現

**分類：** BUG

**相關工單：** 0071（預測功能需求）

**計畫書：** [預測功能計畫書](../../docs/PREDICTION_FEATURE_PLAN.md)

---

## 問題描述

目前問完牌之後，直接換下一個玩家，沒有顯示預測選項。

## 預期行為

問牌完成後，應該顯示預測選項給當前玩家：

```
┌─────────────────────────────────────────┐
│           問牌完成！                      │
│                                         │
│   是否要預測蓋牌中有哪個顏色？            │
│                                         │
│   [紅色]  [黃色]  [綠色]  [藍色]         │
│                                         │
│           [跳過]                         │
└─────────────────────────────────────────┘
```

## 正確流程

```
1. 玩家執行問牌動作
2. 牌轉移完成
3. 【應顯示】預測選項
   - 可選擇一個顏色進行預測
   - 或選擇「跳過」
4. 玩家做出選擇後
5. 輪到下一個玩家
```

## 目前行為

```
1. 玩家執行問牌動作
2. 牌轉移完成
3. 【直接跳過】預測選項 ← 問題在這
4. 輪到下一個玩家
```

## 修復方向

1. 問牌完成後，進入「預測階段」而非直接換人
2. 顯示預測選項介面給當前玩家
3. 等待玩家選擇（預測顏色或跳過）
4. 玩家選擇後才輪到下一個玩家

## 預測選項介面需求

| 元素 | 說明 |
|------|------|
| 四個顏色按鈕 | 紅、黃、綠、藍，點擊即預測該顏色 |
| 跳過按鈕 | 不預測，直接結束回合 |
| 提示文字 | 「是否要預測蓋牌中有哪個顏色？」 |

## 驗收標準

- [ ] 問牌完成後顯示預測選項
- [ ] 可點擊顏色按鈕進行預測
- [ ] 可點擊「跳過」不預測
- [ ] 選擇後才輪到下一個玩家
- [ ] 預測內容公開顯示給所有玩家

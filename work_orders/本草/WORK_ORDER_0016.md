# 工作單 0016

**日期：** 2026-01-23

**工作單標題：** 建立遊戲房間組件 - 基礎結構

**工單主旨：** UI 組件 - 建立遊戲房間（GameRoom）組件的基礎結構

**內容：**

## 工作內容

1. **建立 `frontend/src/components/GameRoom/GameRoom.js` 檔案**

2. **建立組件基礎結構**
   - 建立函數式組件或類別組件
   - 設定基本的 JSX 結構
   - 從 URL 參數或 Redux store 取得遊戲ID

3. **實作基本 UI 佈局**
   - 頂部區域：遊戲標題和當前玩家資訊
   - 中央區域：遊戲桌面（預留給 GameBoard 組件）
   - 左側/右側區域：其他玩家資訊（預留給玩家列表組件）
   - 底部區域：玩家自己的手牌（預留給 PlayerHand 組件）
   - 操作區域：問牌/猜牌按鈕（預留給操作組件）

4. **連接 Redux Store**
   - 使用 `useSelector` 取得遊戲狀態
   - 取得當前玩家資訊
   - 取得遊戲階段資訊

5. **實作基本的生命週期管理**
   - 組件掛載時載入遊戲狀態
   - 組件卸載時清理資源

6. **建立基本的樣式檔案**
   - 建立 `frontend/src/components/GameRoom/GameRoom.css`
   - 設定基本的佈局樣式（使用 Flexbox 或 Grid）
   - 設定響應式設計基礎

7. **匯出組件**
   - 確保組件可以正確匯出和匯入

8. **使用 JSDoc 註解**
   - 為組件添加 JSDoc 註解

## 驗收標準

- [ ] `GameRoom.js` 組件已建立
- [ ] 組件可以正確渲染
- [ ] 基本的 UI 佈局已建立
- [ ] Redux 連接正確
- [ ] 可以正確取得遊戲狀態
- [ ] 基本的樣式已設定
- [ ] 組件有完整的 JSDoc 註解

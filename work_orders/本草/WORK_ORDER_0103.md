## 測試場景

### 場景 1：基本預測流程

1. 玩家 A 執行問牌
2. 問牌完成後顯示預測選項
3. 玩家 A 選擇紅色預測
4. 玩家 A 點擊「結束回合」
5. 輪到玩家 B
6. 所有玩家能看到玩家 A 的預測

**預期結果：**
- 預測記錄正確
- 回合正確切換
- 預測公開廣播

### 場景 2：跳過預測

1. 玩家 A 執行問牌
2. 問牌完成後顯示預測選項
3. 玩家 A 不選顏色，直接點擊「結束回合」
4. 輪到玩家 B

**預期結果：**
- 不記錄預測
- 回合正確切換

### 場景 3：預測結算（猜對）

1. 多名玩家各自問牌並預測
2. 某玩家猜牌成功
3. 顯示預測結算結果

**預期結果：**
- 正確預測 +1 分
- 錯誤預測 -1 分
- 結算畫面正確顯示

### 場景 4：預測結算（0 分邊界）

1. 0 分玩家預測錯誤
2. 驗證分數不會變負

**預期結果：**
- 分數維持 0

### 場景 5：新局清理

1. 完成一局有預測的遊戲
2. 開始新局
3. 驗證預測陣列已清空

**預期結果：**
- predictions 陣列為空

## 執行方式

1. 啟動本地服務
2. 開啟多個瀏覽器視窗模擬多玩家
3. 依序執行測試場景
4. 記錄測試結果

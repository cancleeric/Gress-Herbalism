**編號**：0177

**工作單標題**：功能驗證測試 — 玩家資料與好友系統

**工單主旨**：對工單 0174-0176 修復後的功能進行完整驗證測試

---

## 內容

### 背景

工單 0174-0176 修復了玩家資料保存、Profile 頁面、好友線上狀態等問題。本工單執行完整的功能驗證測試。

### 工作內容

#### 1. 玩家資料功能驗證

| 測試項目 | 驗證方式 |
|---------|---------|
| Google 登入同步 | 登入後確認 Supabase players 表有記錄 |
| 遊戲記錄保存 | 完成一場遊戲後確認 game_history、game_participants 資料正確 |
| player_id 關聯 | 確認 game_participants.player_id 非 NULL |
| 統計更新 | 確認 players 表的 games_played、games_won 正確增加 |
| Profile 顯示 | Google 登入玩家看到正確的統計數據和歷史紀錄 |
| 匿名玩家阻止 | 訪客無法存取 Profile 頁面 |

#### 2. 好友功能驗證

| 測試項目 | 驗證方式 |
|---------|---------|
| 搜尋 Google 玩家 | 用顯示名稱搜尋能找到 |
| 加好友流程 | 發送請求 → 對方接受 → 雙方好友列表互相出現 |
| 刪除好友 | 刪除後雙方列表都移除 |
| 線上狀態 | 好友連線時顯示綠點，斷線時顯示灰點 |
| 遊戲邀請 | 發送邀請 → 對方接受 → 返回 room_id |

#### 3. 既有測試執行

| 測試 | 標準 |
|------|------|
| 後端測試 | 190/190 通過 |
| 前端 GameRoom 測試 | 61/61 通過 |
| 前端 Profile 測試 | 全部通過 |
| 前端 Friends 測試 | 全部通過 |

### 驗收標準

| 標準 | 說明 |
|------|------|
| 玩家資料測試全通過 | A 類所有測試項目通過 |
| 好友功能測試全通過 | B 類所有測試項目通過 |
| 既有測試不受影響 | 所有既有測試通過 |

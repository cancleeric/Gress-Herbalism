# 工單 0133：猜牌結果對話框 UI 重新設計

## 目標
根據 Stitch 生成的設計稿，重新設計猜牌結果對話框（猜對了/猜錯了），延續中國風草藥主題設計風格。

## 遊戲邏輯說明
- 猜牌後顯示結果（猜對了/猜錯了）
- 顯示蓋牌的實際顏色
- 顯示所有玩家的分數變化
- 顯示目前分數排名
- 遊戲未結束：顯示「下一局」按鈕
- 遊戲結束：顯示「恭喜 XXX 獲勝！」和「離開房間」按鈕

## 設計規格

### 整體佈局
- Modal 寬度：max-w-[400px]
- 米白色背景（#F5F1E6 off-white）+ 金色邊框
- 圓角 lg，陰影效果
- 中國風紋理背景（chinese-pattern-light）

### 頂部區域
- 圖示：badge（猜對）或 close（猜錯）
- 標題：「猜對了！」（綠色）或「猜錯了！」（紅色）
- 文字大小：32px，粗體

### 蓋牌顯示區
- 標題：「蓋牌是：」（金色）
- 顏色標籤（pill 樣式）：
  - 紅色 (#E53E3E) - 火 (Red) + 圖示 local_fire_department
  - 黃色 (#D69E2E) - 土 (Yellow) + 圖示 energy_savings_leaf
  - 綠色 (#43A047) - 木 (Green) + 圖示 eco
  - 藍色 (#1E88E5) - 水 (Blue) + 圖示 water_drop

### 分數變化區塊
- 標題：「分數變化」+ analytics 圖示
- 卡片樣式列表，背景 #efeadc（sub-card）
- 每個玩家顯示：
  - 頭像圖示
  - 名稱 + 角色標籤（猜牌者/跟猜）
  - 說明文字（猜對獲得加分/未猜測）
  - 分數變化（+3 綠色，-1 紅色）

### 目前分數區塊
- 標題：「目前分數」+ leaderboard 圖示
- 排名列表，依分數高低排序
- 顯示排名數字
- 達到 7 分者顯示「勝利！」標籤（紅色背景）

### 底部區域
- 遊戲未結束：「下一局」按鈕（綠色）+ navigate_next 圖示
- 遊戲結束：
  - 「恭喜 XXX 獲勝！」文字
  - 「離開房間」按鈕

### 設計風格
- 配色：
  - off-white: #F5F1E6（背景）
  - herbal-gold: #bb9c63（金色）
  - deep-green: #2f7f34（綠色）
  - soft-red: #e83b3e（紅色）
  - sub-card: #efeadc（卡片背景）
- Material Symbols 圖示
- 圓角、陰影效果

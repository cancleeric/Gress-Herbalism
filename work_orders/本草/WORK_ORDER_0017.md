# 工作單 0017

**日期：** 2026-01-23

**工作單標題：** 建立遊戲桌面組件 - 蓋牌顯示

**工單主旨：** UI 組件 - 建立遊戲桌面（GameBoard）組件顯示蓋牌

**內容：**

## 工作內容

1. **建立 `frontend/src/components/GameBoard/GameBoard.js` 檔案**

2. **建立組件基礎結構**
   - 建立函數式組件
   - 接收遊戲狀態作為 props（或從 Redux 取得）

3. **實作蓋牌顯示**
   - 顯示兩張蓋牌的位置
   - 蓋牌顯示為背面（不顯示顏色）
   - 使用卡片樣式顯示蓋牌
   - 蓋牌應該有視覺效果（例如：卡片背面圖案、陰影效果）

4. **實作蓋牌動畫效果**（可選，但建議實作）
   - 蓋牌出現動畫
   - 蓋牌翻轉效果（當遊戲結束時顯示正面）

5. **連接 Redux Store**
   - 使用 `useSelector` 取得 `hiddenCards`、`gamePhase`、當前玩家資訊
   - **猜牌時：顯示正確答案給猜牌者查看（蓋牌的實際顏色）**
   - **猜對時：公布正確答案給所有玩家（顯示蓋牌的實際顏色）**
   - **猜錯時：蓋牌保持隱藏狀態（不顯示顏色）**
   - 遊戲結束時（有獲勝者或沒有獲勝者），根據情況顯示或隱藏蓋牌

6. **建立樣式檔案**
   - 建立 `frontend/src/components/GameBoard/GameBoard.css`
   - 設定蓋牌的樣式
   - 設定蓋牌的佈局（兩張牌並排或上下排列）
   - 設定卡片背面圖案或顏色

7. **匯出組件**
   - 確保組件可以正確匯出和匯入

8. **使用 JSDoc 註解**
   - 為組件添加 JSDoc 註解

# 工作單 0015

**日期：** 2026-01-23

**工作單標題：** 建立遊戲大廳組件 - 創建和加入房間功能

**工單主旨：** UI 組件 - 實作遊戲大廳的創建和加入房間功能

**內容：**

## 工作內容

1. **在 `frontend/src/components/Lobby/Lobby.js` 中新增功能**

2. **實作創建房間功能**
   - `handleCreateRoom()` 函數
   - 收集玩家資訊（玩家名稱、玩家數量選擇）
   - 驗證玩家數量（3-4人）
   - 調用 `gameService.createGame()` 創建遊戲
   - 更新 Redux store 或本地狀態
   - 導航到遊戲房間頁面

3. **實作加入房間功能**
   - `handleJoinRoom()` 函數
   - 接收房間ID輸入
   - 驗證房間ID格式
   - 調用 `gameService.getGameState()` 取得遊戲狀態
   - 驗證房間是否還有空位
   - 加入玩家到遊戲
   - 更新 Redux store 或本地狀態
   - 導航到遊戲房間頁面

4. **實作玩家資訊輸入**
   - 玩家名稱輸入框
   - 玩家數量選擇器（3人或4人）
   - 表單驗證

5. **實作錯誤處理**
   - 顯示創建房間失敗的錯誤訊息
   - 顯示加入房間失敗的錯誤訊息（房間不存在、房間已滿等）

6. **連接 Redux**
   - 使用 `useSelector` 取得遊戲狀態
   - 使用 `useDispatch` 發送 actions

7. **使用 JSDoc 註解**
   - 為所有函數添加 JSDoc 註解

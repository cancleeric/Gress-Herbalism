# 工作單 0020

**日期：** 2026-01-23

**工作單標題：** 建立問牌介面組件 - 表單驗證和提交

**工單主旨：** UI 組件 - 實作問牌介面的表單驗證和提交功能

**內容：**

## 工作內容

1. **在 `frontend/src/components/QuestionCard/QuestionCard.js` 中新增功能**

2. **實作表單驗證**
   - 驗證顏色選擇：必須選擇兩個不同顏色
   - 驗證目標玩家選擇：必須選擇一個目標玩家
   - 驗證要牌方式選擇：必須選擇一種要牌方式
   - 使用 `gameRules.js` 中的驗證函數驗證問牌動作是否有效
   - 顯示驗證錯誤訊息

3. **實作問牌動作提交**
   - `handleSubmit()` 函數
   - 收集表單資料（顏色、目標玩家、要牌方式）
   - 建立問牌動作物件
   - 調用 `gameService.processQuestionAction()` 處理問牌動作
   - 更新 Redux store
   - 關閉問牌介面

4. **實作問牌結果顯示**
   - 顯示問牌結果（成功或失敗）
   - 顯示收到的牌（如果成功）
   - 顯示錯誤訊息（如果失敗）

5. **連接 Redux Store**
   - 使用 `useSelector` 取得遊戲狀態和玩家資訊
   - 使用 `useDispatch` 發送問牌動作

6. **實作取消功能**
   - `handleCancel()` 函數
   - 關閉問牌介面
   - 清除表單狀態

7. **實作載入狀態**
   - 提交時顯示載入指示器
   - 防止重複提交

8. **使用 JSDoc 註解**
   - 為所有函數添加 JSDoc 註解

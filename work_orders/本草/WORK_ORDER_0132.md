# 工單 0132：遊戲結束畫面 UI 統一

## 目標
將遊戲結束畫面改為與遊戲進行中相同的 UI 結構，保持視覺一致性。

## 遊戲邏輯說明
- 遊戲結束後，畫面應該與遊戲進行中一樣顯示三欄式佈局
- 遊戲狀態顯示為「遊戲結束」
- 跳出「猜對了」對話框顯示結果（此部分後續工單處理）

## 設計規格

### 整體佈局
- 使用與遊戲進行中相同的 HTML/CSS 結構
- 三欄式佈局：遊戲紀錄 | 遊戲桌面 | 玩家列表
- Header 顯示「遊戲結束」狀態

### 狀態顯示
- 將 `GAME_PHASE_FINISHED` 加入遊戲進行中的渲染條件
- Header 狀態文字根據 `gamePhase` 動態顯示：
  - `GAME_PHASE_PLAYING` → 「遊戲進行中」
  - `GAME_PHASE_FINISHED` → 「遊戲結束」

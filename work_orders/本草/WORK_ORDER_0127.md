# 工單 0127：問牌選擇面板 UI 重新設計

## 目標
根據 Stitch 生成的設計稿，重新設計問牌選擇面板（QuestionFlow），延續中國風草藥主題設計風格。

## 設計規格

### 整體佈局
- 寬版 Modal（max-w-5xl）
- 三欄式佈局，三個步驟並排顯示
- 米白色背景（#f5f1e6）+ 金色邊框
- 水彩風格裝飾背景

### Header
- 標題「問牌」+ 英文「(Inquiry)」
- 右側顯示當前顏色組合（圓形色塊 chips）

### 三欄內容

#### 步驟 1：選擇目標玩家
- 金色圓形步驟編號
- 標題「選擇目標玩家」
- 玩家列表（radio button 樣式）
- 每個選項：玩家名稱 + 位置描述
- hover 時金色邊框高亮

#### 步驟 2：選擇要牌方式
- 標題「選擇要牌方式」
- 三個按鈕選項：
  - 各一張 (Each Color)
  - 其中一種全部 (All of One)
  - 給一張要全部 (Give & Take)
- 選中時綠色邊框 + 勾勾圖示
- 步驟 2.5：給牌顏色選擇（條件顯示）

#### 步驟 3：確認問牌內容
- 確認卡片（虛線邊框）
- 顯示：對象、行動、說明
- 底部提示文字

### Footer
- 左側：取消按鈕（灰色）
- 右側：確認問牌按鈕（綠色 + send 圖示）

### 設計風格
- 配色：primary #2f7f34、gold #bb9c63、parchment #f5f1e6
- 字體：Noto Serif TC（標題）、Noto Sans TC（內文）
- Material Symbols 圖示
- 圓角、陰影效果
- 草藥裝飾元素（eco、psychiatry 圖示）

### 檔案變更

7. 點擊確認可提交問牌

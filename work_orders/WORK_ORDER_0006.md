# 工作單 0006

**日期：** 2026-01-23

**工作單標題：** 建立牌組工具函數 - 手牌管理輔助函數

**工單主旨：** 工具函數 - 建立手牌管理的輔助函數

**內容：**

## 工作內容

1. **在 `frontend/src/utils/cardUtils.js` 中新增函數**

2. **實作 `getCardsByColor(hand, color)` 函數**
   - 從手牌中篩選出指定顏色的牌
   - 接收手牌陣列和顏色作為參數
   - 返回該顏色的牌陣列

3. **實作 `hasCard(hand, cardId)` 函數**
   - 檢查手牌中是否包含指定的牌
   - 接收手牌陣列和牌ID作為參數
   - 返回布林值

4. **實作 `removeCard(hand, cardId)` 函數**
   - 從手牌中移除指定的牌
   - 接收手牌陣列和牌ID作為參數
   - 返回新的手牌陣列（不修改原陣列）

5. **實作 `addCard(hand, card)` 函數**
   - 將牌加入手牌
   - 接收手牌陣列和牌物件作為參數
   - 返回新的手牌陣列（不修改原陣列）

6. **實作 `countCardsByColor(hand, color)` 函數**
   - 計算手牌中指定顏色的牌數
   - 接收手牌陣列和顏色作為參數
   - 返回數量

7. **使用 JSDoc 註解**
   - 為所有函數添加完整的 JSDoc 註解

- [ ] 函數可以正確操作手牌

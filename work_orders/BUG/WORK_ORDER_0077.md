# 工作單 0077

**日期：** 2026-01-25

**工作單標題：** 移除顏色牌上的「你」字樣

**工單主旨：** BUG 修復 - 顏色牌禁用標記不應顯示「你」字樣

**分類：** BUG

---

## 問題描述

顏色牌上顯示了「你」的字樣標記，這個標記不應該出現。

## 錯誤截圖

紅藍顏色牌上顯示了「你」的標記。

## 預期行為

顏色牌上不應顯示「你」字樣。

如需標示禁用狀態，應使用其他方式：
- 灰色遮罩
- 半透明效果
- 禁止圖示
- 或不顯示任何標記

## 修復方向

移除顏色牌上的「你」字樣顯示邏輯。

## 驗收標準

- [ ] 顏色牌上不顯示「你」字樣
- [ ] 禁用的牌可用其他視覺效果表示（如灰色）

# 工作單 0022

**日期：** 2026-01-23

**工作單標題：** 建立遊戲狀態顯示組件

**工單主旨：** UI 組件 - 建立遊戲狀態顯示（GameStatus）組件

**內容：**

## 工作內容

1. **建立 `frontend/src/components/GameStatus/GameStatus.js` 檔案**

2. **建立組件基礎結構**
   - 建立函數式組件
   - 從 Redux store 取得遊戲狀態

3. **實作當前玩家顯示**
   - 顯示當前輪到哪個玩家
   - 當前玩家要有視覺高亮

4. **實作玩家狀態顯示**
   - 顯示所有玩家的狀態（活躍/已退出）
   - 顯示每個玩家的名稱
   - 已退出的玩家要有視覺標記（例如：灰色、刪除線）

5. **實作遊戲階段顯示**
   - 顯示當前遊戲階段（等待中/進行中/已結束）
   - 如果遊戲已結束，顯示獲勝者

6. **實作遊戲歷史記錄顯示**（可選，但建議實作）
   - 顯示最近的遊戲動作歷史
   - 顯示問牌和猜牌的記錄
   - 歷史記錄可以滾動查看

7. **連接 Redux Store**
   - 使用 `useSelector` 取得遊戲狀態
   - 當遊戲狀態更新時自動更新顯示

8. **建立樣式檔案**
   - 建立 `frontend/src/components/GameStatus/GameStatus.css`
   - 設定狀態顯示的佈局樣式
   - 設定不同狀態的視覺樣式

9. **匯出組件**
   - 確保組件可以正確匯出和匯入

10. **使用 JSDoc 註解**
    - 為組件添加 JSDoc 註解

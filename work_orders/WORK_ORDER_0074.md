## 功能概述

重構問牌流程，將原本「手動選擇兩種顏色」改為「從桌面點擊顏色組合牌」。

## 流程變更

### 原本流程

```
1. 選擇「問牌」
2. 手動選擇第一種顏色
3. 手動選擇第二種顏色
4. 選擇目標玩家
5. 選擇要牌方式
```

### 新流程

```
1. 從桌面中央點擊一張顏色組合牌
2. 選擇目標玩家
3. 選擇要牌方式
4. 確認問牌
```

## 詳細流程

### 步驟 1：點擊顏色組合牌

```
┌─────────────────────────────────────────────────────────┐
│                      遊戲桌面                            │
│                                                         │
│              ┌─────────────────────┐                    │
│              │    選擇要問的顏色    │                    │
│              │                     │                    │
│              │  [紅綠] [綠藍] [綠黃] │  ← 可點擊的牌     │

│              │                     │                    │
│              └─────────────────────┘                    │
│                                                         │
│     提示：點擊一張顏色牌開始問牌                          │
└─────────────────────────────────────────────────────────┘
```

### 步驟 2：選擇目標玩家

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│     已選擇：[紅綠] 紅色 + 綠色                           │
│                                                         │
│     選擇要問牌的對象：                                   │
│                                                         │
│     [玩家2]  [玩家3]  [玩家4]                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 步驟 3：選擇要牌方式

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│     已選擇：[紅綠] → 玩家2                               │
│                                                         │
│     選擇要牌方式：                                       │
│                                                         │
│     [各一張]  [全部]  [其中一種全部]                     │
│                                                         │
│                    [確認問牌]                            │
└─────────────────────────────────────────────────────────┘
```

## 介面互動

### 顏色牌狀態

| 狀態 | 外觀 | 說明 |
|------|------|------|
| 可選擇 | 正常顯示 | 可以點擊 |
| 已選擇 | 高亮/邊框 | 當前選中的牌 |
| 禁用 | 灰色/半透明 | 上回合選過，不能選 |
| Hover | 放大/陰影 | 滑鼠懸停效果 |

### 點擊行為

```
點擊顏色牌：
├─ 如果是禁用狀態 → 顯示提示「不能選擇上回合選過的牌」
├─ 如果已選擇其他牌 → 切換選擇
└─ 正常情況 → 選中該牌，進入下一步
```

## 技術實作

### 1. 遊戲狀態擴充

```javascript
const gameState = {
  // ... 現有欄位

  // 顏色牌選擇記錄
  colorCardSelections: {
    // playerId: 上回合選擇的顏色牌 ID
    'player1': 'red-green',
    'player2': null,
    // ...
  },

  // 當前問牌流程
  currentQuestion: {
    selectedCard: null,      // 選擇的顏色牌
    targetPlayer: null,      // 目標玩家
    askType: null,           // 要牌方式
  },
};
```

### 2. Socket 事件

```javascript
// 前端 → 後端
'selectColorCard'     // 選擇顏色牌 { cardId: 'red-green' }
'selectTargetPlayer'  // 選擇目標玩家 { playerId: 'player2' }
'selectAskType'       // 選擇要牌方式 { askType: 'oneEach' }
'confirmQuestion'     // 確認問牌

// 後端 → 前端
'colorCardSelected'   // 廣播：某玩家選了哪張顏色牌
'questionConfirmed'   // 問牌確認，執行牌轉移
```

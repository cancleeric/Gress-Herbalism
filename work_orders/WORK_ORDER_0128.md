# 工單 0128：創建房間面板 UI 重新設計

## 目標
根據 Stitch 生成的設計稿，重新設計創建房間面板（Create Room Modal），延續中國風草藥主題設計風格。

## 設計規格

### 整體佈局
- Modal 寬度：max-w-[400px]
- 米白色背景（#f5f1e6）+ 金色邊框
- 圓角 xl，陰影效果
- 水彩風格裝飾背景（eco、psychiatry 圖示）

### 頂部區域
- 圖示：add_circle（綠色，5xl）
- 標題「創建新房間」（粗體，2xl）
- 副標題「設定房間參數後開始遊戲」（灰色，sm）

### 表單區域

#### 玩家數量
- 標籤 + 下拉選單
- 自訂樣式選單（金色邊框）
- 選項：3人 / 4人

#### 設為私人房間
- checkbox + 標籤 + 鎖頭圖示
- 選中時顯示密碼輸入框

#### 房間密碼（條件顯示）
- 標籤 + 密碼輸入框
- 右側鑰匙圖示
- placeholder：請輸入密碼

### 按鈕區域
- 取消按鈕：灰色背景
- 創建房間按鈕：綠色背景 + done_all 圖示
- 兩按鈕等寬（flex-1）

### 底部裝飾
- 金色橫條（h-1.5）

### 設計風格
- 配色：primary #2f7f34、gold #bb9c63、parchment #f5f1e6
- 字體：Noto Serif TC（標題）、Noto Sans TC（內文）
- Material Symbols 圖示
- 圓角、陰影效果
- 草藥裝飾元素

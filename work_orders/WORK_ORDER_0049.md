## 背景說明

需要在前端新增計分板顯示和跟猜決定介面。

## 需要實作的組件

### 1. 計分板組件 (Scoreboard)

**檔案：** `frontend/src/components/Scoreboard/Scoreboard.js`

**功能：**
- 顯示所有玩家的當前分數
- 標示領先玩家
- 顯示距離勝利分數（7分）的差距
- 當前局數顯示

```

└─────────────────────────┘
```

### 2. 跟猜決定介面 (FollowGuessPanel)

**檔案：** `frontend/src/components/FollowGuessPanel/FollowGuessPanel.js`

**功能：**
- 顯示猜牌玩家的猜測內容
- 讓當前玩家選擇「跟猜」或「不跟猜」
- 顯示跟猜的風險提示（跟對 +1，跟錯 -1）
- 顯示其他玩家的決定狀態

**UI 設計：**
```
┌─────────────────────────────────────┐
│         跟猜決定                     │
├─────────────────────────────────────┤
│ 小明 猜測蓋牌是：                    │
│        [紅色] [藍色]                 │
├─────────────────────────────────────┤
│ 你要跟猜嗎？                        │
│ ・跟對：+1 分                        │
│ ・跟錯：-1 分（並退出當局）           │
├─────────────────────────────────────┤
│   [跟猜]        [不跟猜]            │
├─────────────────────────────────────┤
│ 其他玩家：                          │
│ 小華：等待中...                      │
│ 小美：已決定                         │
└─────────────────────────────────────┘
```

### 3. 局結束畫面 (RoundEndScreen)

**檔案：** `frontend/src/components/RoundEndScreen/RoundEndScreen.js`

**功能：**
- 顯示當局結果（誰猜對/無人猜對）
- 顯示蓋牌答案
- 顯示本局分數變化
- 顯示當前總分
- 「開始下一局」按鈕或自動倒數

**UI 設計：**
```
┌─────────────────────────────────────┐
│         第 3 局結束                  │
├─────────────────────────────────────┤
│ 🎉 小明 猜對了！                     │
│ 正確答案：[紅色] [藍色]              │
├─────────────────────────────────────┤
│ 本局得分：                          │
│ 小明：+3 (猜對)                      │
│ 小華：+1 (跟猜正確)                  │
│ 小美：-1 (跟猜錯誤)    ← 不會顯示，因為猜對了│
├─────────────────────────────────────┤
│ 當前總分：                          │
│ 小明：5分  小華：4分  小美：2分      │
├─────────────────────────────────────┤
│     [開始第 4 局] (5秒後自動開始)    │
└─────────────────────────────────────┘
```

### 4. 遊戲結束畫面 (GameEndScreen)

**檔案：** `frontend/src/components/GameEndScreen/GameEndScreen.js`

- 顯示最終贏家

- 顯示遊戲統計（總局數等）
- 「再來一場」和「返回大廳」按鈕

### 6. Socket 事件處理

更新 socket 事件監聽：
- `round:end` - 更新局結束狀態
- `round:started` - 更新新局狀態
- `followGuess:request` - 顯示跟猜決定介面
- `followGuess:update` - 更新其他玩家的決定狀態
- `game:won` - 顯示遊戲結束畫面

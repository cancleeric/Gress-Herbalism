# 工單 0124：遊戲進行中階段 UI 重新設計

## 目標
根據 Stitch 生成的設計稿，重新設計遊戲進行中階段（`GAME_PHASE_PLAYING`）的 UI，延續工單 0123 等待階段的設計風格。

## 設計規格

### 整體佈局
三欄式佈局（左-中-右）+ 底部手牌區，與等待階段類似

### Header（頂部導航）
- 左側：品牌標誌「Herbalism 本草」+ 綠色脈動點 + 「Game in Progress」狀態
- 右側：「Leave」按鈕

### 左欄
1. **自己的玩家卡片**
   - 大頭像 + ME 標籤
   - 玩家名稱和等級
   - 分數和手牌數統計

2. **遊戲紀錄面板**
   - 標題「遊戲紀錄」
   - 可滾動的歷史記錄列表
   - 當前玩家的記錄用綠色邊框標示

### 中央區域
1. **蓋牌區（The Hidden Cure）**
   - 標題 + 神秘圖示
   - 兩張蓋牌（綠色背景 + 問號）

2. **顏色組合牌（Inquiry Cards）**
   - 6 張牌排成 2 排 x 3 列
   - 紅+黃、紅+綠、紅+藍
   - 黃+綠、黃+藍、綠+藍
   - hover 時放大效果
   - 點擊開始問牌流程

### 右欄（玩家列表）
- 標題「Players」
- 玩家卡片列表：
  - 當前回合玩家：金色邊框 + 發光效果 + TURN 標籤
  - 其他玩家：淡化顯示
  - 顯示：頭像、名稱、分數、手牌數
- 底部：回合資訊提示

### 底部手牌區（Footer）
- 左側：「My Hand」標籤 + 手牌數量
- 中間：手牌卡片（紅、黃、綠、藍）
  - hover 時向上浮起效果
  - 顏色對應：#E53935、#FDD835、#43A047、#1E88E5
- 右側：問牌、猜牌按鈕（中國風樣式）

### 設計風格
- 配色：
  - 主色：`#bb9c63`（金色）
  - 草藥綠：`#2f7f34`
  - 草藥金：`#C5A059`
  - 背景：`#f5f1e6`（淺色）/ `#1d1a15`（深色）
- 字體：Be Vietnam Pro、Noto Serif TC
- 圓角卡片、陰影效果
- 當前回合發光效果（turn-glow）

## 技術實作

### 檔案變更
1. **GameRoom.js** - 新增遊戲進行中專用 UI
2. **GameRoom.css** - 新增遊戲進行中樣式
3. **GameRoom.test.js** - 更新測試

### 實作重點
1. 當 `gamePhase === GAME_PHASE_PLAYING` 時渲染新 UI
2. 複用等待階段的 Header 結構，調整狀態顯示
3. 整合現有的：
   - 顏色組合牌點擊 → 問牌流程
   - 猜牌按鈕 → 猜牌流程
   - 遊戲紀錄顯示
4. 當前回合玩家高亮顯示
5. 手牌 hover 效果

### 響應式設計
- 桌面版（≥1024px）：三欄 + 底部
- 平板版（768px-1023px）：調整欄寬
- 手機版（<768px）：單欄垂直堆疊，底部固定

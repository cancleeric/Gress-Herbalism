# 工作單 0028

**日期：** 2026-01-23

**工作單標題：** 建立基礎樣式系統

**工單主旨：** 樣式設計 - 建立專案的基礎樣式系統

**內容：**

## 工作內容

1. **建立 `frontend/src/styles/` 目錄結構**
   - `variables.css` - CSS 變數定義
   - `base.css` - 基礎樣式
   - `components.css` - 組件通用樣式
   - `utilities.css` - 工具類樣式

2. **建立 CSS 變數**
   - 定義顏色變數（對應遊戲的四種顏色）
   - 定義間距變數
   - 定義字體變數
   - 定義陰影變數
   - 定義過渡動畫變數

3. **建立基礎樣式**
   - 重置樣式（reset.css）
   - 基礎字體設定
   - 基礎佈局設定
   - 基礎按鈕樣式
   - 基礎輸入框樣式

4. **建立組件通用樣式**
   - 卡片樣式
   - 按鈕樣式變體
   - 輸入框樣式變體
   - Modal/Dialog 樣式

5. **建立工具類樣式**
   - 佈局工具類（flex, grid 等）
   - 間距工具類
   - 文字工具類
   - 顏色工具類

6. **建立響應式設計基礎**
   - 定義斷點（breakpoints）
   - 建立響應式工具類

7. **在主應用中引入樣式**
   - 在 `App.js` 或 `index.js` 中引入樣式檔案

## 驗收標準

- [ ] 樣式目錄結構已建立
- [ ] CSS 變數已定義
- [ ] 基礎樣式已建立
- [ ] 組件通用樣式已建立
- [ ] 工具類樣式已建立
- [ ] 響應式設計基礎已建立
- [ ] 樣式已正確引入到應用中

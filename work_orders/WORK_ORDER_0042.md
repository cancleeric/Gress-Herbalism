# 工作單 0042

**日期：** 2026-01-23

**工作單標題：** ngrok 部署測試與驗證

**工單主旨：** 測試驗證 - 完整測試 ngrok 環境下的多人遊戲功能

**內容：**

## 背景說明

在 ngrok 環境設定完成後，需要完整測試所有多人遊戲功能，確保外部玩家可以正常遊玩。

## 測試環境準備

1. **本機環境**
   - 電腦瀏覽器（玩家 1）

2. **外部環境**（至少選一種）
   - 手機瀏覽器（使用行動網路，非 WiFi）
   - 另一台電腦
   - 請同學從他們的網路連線

## 測試項目

### 1. 基本連線測試
- [ ] 外部裝置可以開啟前端 ngrok URL
- [ ] 頁面正常載入，無錯誤
- [ ] Console 無 CORS 或連線錯誤

### 2. 房間功能測試
- [ ] 玩家 1 可以創建房間
- [ ] 玩家 2（外部）可以看到可用房間列表
- [ ] 玩家 2 可以透過房間 ID 加入房間
- [ ] 玩家 2 可以透過點擊房間列表加入

### 3. 即時同步測試
- [ ] 玩家加入/離開時，其他玩家即時收到通知
- [ ] 房間列表即時更新
- [ ] 遊戲狀態即時同步

### 4. 遊戲流程測試
- [ ] 遊戲可以正常開始
- [ ] 問牌功能正常運作
- [ ] 猜牌功能正常運作
- [ ] 手牌變化即時同步
- [ ] 遊戲結束正常顯示結果

### 5. 網路穩定性測試
- [ ] 短暫斷線後可以重新連線
- [ ] 長時間遊戲連線穩定

## 常見問題排查

| 問題 | 可能原因 | 解決方式 |
|------|----------|----------|
| 頁面無法載入 | ngrok 未啟動 | 確認 ngrok 正在運行 |
| API 請求失敗 | CORS 設定錯誤 | 檢查後端 CORS 設定 |
| WebSocket 連不上 | Socket.io CORS 設定錯誤 | 檢查 Socket.io CORS |
| 連上但無法同步 | 前端連到錯誤的後端 URL | 檢查 .env 設定 |

## 驗收標準

- [ ] 所有基本連線測試通過
- [ ] 所有房間功能測試通過
- [ ] 所有即時同步測試通過
- [ ] 所有遊戲流程測試通過
- [ ] 至少一位外部玩家成功完成一場完整遊戲

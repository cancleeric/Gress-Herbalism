# 工單 0131：跟猜決定面板 UI 重新設計（修正版）

## 目標
根據 Stitch 生成的設計稿，重新設計跟猜決定面板，延續中國風草藥主題設計風格。

**注意：此工單修正 0130 的錯誤實作。問題是新面板被添加到錯誤的位置，而非替換原有面板。**

## 遊戲邏輯說明
- 有玩家猜牌後，其他玩家按順序決定是否跟猜
- 跟對 +1 分，跟錯 -1 分並退出當局
- 玩家按順序決定，輪到自己時才能點擊按鈕
- 顯示猜牌者猜測的兩個顏色
- 顯示所有玩家的決定狀態

## 設計規格

### 整體佈局
- Modal 寬度：max-w-[500px]
- 米白色背景（#F5F1E6 parchment）+ 金色邊框
- 圓角 lg，陰影效果
- 草藥紋理背景（herbal-texture）
- 角落裝飾（金色邊角）

### 頂部區域
- 標題「跟猜階段」（粗體，3xl，font-display）
- 紅色底線裝飾（traditional-red #B22222）

### 猜牌資訊區塊
- 白色半透明背景（bg-white/50）
- 文字：「[玩家名稱] 猜測蓋牌是：」
- 顏色標籤（pill 樣式）：
  - 紅色 (#E53E3E) - 「紅色藥草」
  - 黃色 (#D69E2E) - 「黃色藥草」
  - 綠色 (#43A047) - 「綠色藥草」
  - 藍色 (#1E88E5) - 「藍色藥草」

### 玩家決策狀態列表
- 標題：「玩家決策狀態」
- 列表項目（垂直排列，卡片樣式）：
  - 猜牌者：金色邊框 + 「猜牌者」標籤（金色背景）+ 「發起猜測」狀態
  - 已跟猜玩家：「跟隨」標籤（綠色）
  - 當前決定者：橙色邊框高亮 + 「決定中...」標籤（橙色）+ 放大效果
  - 已不跟玩家：「不跟」標籤（灰色）
  - 等待中玩家：半透明 + 「等待中」文字（灰色斜體）
- 每個玩家顯示頭像和名稱

### 決定按鈕區（輪到自己時顯示）
- 提示文字：「輪到你決定！」
- 說明：「跟對 +1 分，跟錯 -1 分 並退出當局」
- 兩個按鈕並排（sm 以上）或堆疊（手機）：
  - 「不跟」按鈕：灰色邊框樣式 + cancel 圖示
  - 「跟猜」按鈕：綠色背景 + check_circle 圖示

### 等待狀態文字
- 猜牌者：「等待其他玩家按順序決定是否跟猜...」
- 已決定：「你選擇了「跟猜/不跟」，等待其他玩家...」
- 未輪到：「還沒輪到你，請等待...」

### 設計風格
- 配色：
  - primary: #2f7f34（綠色主色）
  - gold: #D4AF37（金色點綴）
  - parchment: #F5F1E6（米白背景）
  - traditional-red: #B22222（紅色強調）
- 字體：Noto Serif TC（標題）、Noto Sans TC（內文）
- Material Symbols 圖示
- 圓角、陰影效果

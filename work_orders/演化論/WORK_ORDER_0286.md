## 工作單標題
演化論遊戲開始功能驗證測試

## 工單主旨
測試驗證 - 演化論遊戲無法開始 BUG 修復

## 內容

### 測試項目

1. **房主識別測試**
   - [ ] 房主創建房間後看到「開始遊戲」按鈕
   - [ ] 房主看到提示「您是房主，當所有玩家準備後即可開始遊戲」
   - [ ] 房主卡片顯示「房主」標記

2. **非房主識別測試**
   - [ ] 非房主加入房間後看到「準備」按鈕
   - [ ] 非房主看到提示「點擊「準備」按鈕表示您已準備好開始遊戲」
   - [ ] 非房主卡片不顯示「房主」標記

3. **準備功能測試**
   - [ ] 點擊「準備」按鈕後狀態變為「已準備」
   - [ ] 點擊「取消準備」按鈕後狀態恢復為「未準備」
   - [ ] 其他玩家可以看到準備狀態變化

4. **開始遊戲測試**
   - [ ] 玩家數量不足時，開始按鈕顯示「需要至少 2 位玩家」
   - [ ] 有玩家未準備時，開始按鈕顯示「等待玩家準備」
   - [ ] 所有玩家準備後，點擊「開始遊戲」可以正常開始

5. **離開房間測試**
   - [ ] 點擊「離開房間」可以正常離開
   - [ ] 房主離開後，房主權限正確轉移
   - [ ] 所有玩家離開後，房間正確刪除

6. **玩家標記測試**
   - [ ] 自己的玩家卡片有「我」標記
   - [ ] 房主的玩家卡片有「房主」標記
   - [ ] current-player 樣式正確應用

### 測試步驟

1. 啟動後端伺服器
2. 開啟兩個瀏覽器視窗，登入不同帳號
3. 第一個帳號創建房間
4. 第二個帳號加入房間
5. 依序測試上述項目

### 驗收標準
- [ ] 所有測試項目通過
- [ ] 無 console 錯誤
- [ ] UI 顯示正確

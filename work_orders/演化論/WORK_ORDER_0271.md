# CLAUDE.md

## 專案概述

這是一個多人線上桌遊平台，目前支援兩款遊戲：
- **本草 Herbalism**：3-4 人推理桌遊（已完成）
- **演化論 Evolution**：2-4 人策略卡牌遊戲（開發中）

## 架構概述

### 前端
```
frontend/src/components/
├── common/              # 共用組件（登入、大廳、好友等）
└── games/
    ├── herbalism/       # 本草遊戲組件
    └── evolution/       # 演化論遊戲組件
```

### 後端
```
backend/logic/
├── herbalism/           # 本草遊戲邏輯
└── evolution/           # 演化論遊戲邏輯
```

## 演化論遊戲常數

定義在 `shared/constants/evolution.js`：
- 玩家數量：2-4人
- 卡牌總數：84張
- 遊戲階段：演化 / 食物供給 / 進食 / 滅絕
- 性狀類型：19種
- 計分：生物 +2分、性狀 +1分

## 演化論遊戲規則重點

1. **雙面卡系統**：每張卡可當生物或性狀使用
2. **四階段回合**：演化 → 食物供給 → 進食 → 滅絕
3. **肉食攻擊**：肉食生物必須攻擊其他生物獲得食物
4. **互動性狀**：溝通、合作、共生需連結兩隻生物
5. **遊戲結束**：牌庫空後的最後一回合結束時計分

詳細規則請參考 `docs/演化論/GAME_RULES_EVOLUTION.md`
```

#### 2. README.md 更新

```markdown
# 多人線上桌遊平台

## 支援遊戲

### 🌿 本草 Herbalism
3-4 人推理遊戲。透過問牌和推理猜測蓋牌顏色。

### 🦎 演化論 Evolution
2-4 人策略遊戲。創造生物、賦予性狀，在弱肉強食的世界中生存。

## 快速開始

### 前端
```bash
cd frontend
npm install
npm start
```

### 後端
```bash
cd backend
npm install
npm run dev
```

## 技術棧

- **前端**：React 18 + Redux Toolkit
- **後端**：Node.js + Express + Socket.io
- **資料庫**：Supabase (PostgreSQL)
- **部署**：Google Cloud Run
```

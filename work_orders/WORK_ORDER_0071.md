# 工作單 0071

**日期：** 2026-01-25

**工作單標題：** 新增「預測」功能

**工單主旨：** 功能開發 - 問牌後可選擇預測蓋牌顏色

---

## 功能概述

在現有的遊戲流程中，玩家有「問牌」和「猜牌」兩種動作。本工單新增「預測」功能，讓玩家在問牌後可以選擇預測蓋牌的顏色。

## 功能需求

### 1. 預測時機與回合結束流程

**問牌之後**，玩家可以選擇是否進行預測，**必須按下「結束」按鈕才會換下一位玩家**。

```
遊戲流程：
┌─────────────────────────────────────────────────────┐
│  輪到玩家行動                                         │
│  ├─ 選擇「問牌」 → 問牌流程                           │
│  │                    │                              │
│  │                    ▼                              │
│  │         ┌─────────────────────────┐               │
│  │         │  是否要預測？            │               │
│  │         │                         │               │
│  │         │  [紅] [黃] [綠] [藍]    │  ← 選擇顏色   │
│  │         │       或不選擇          │               │
│  │         │                         │               │
│  │         │      [結束回合]         │  ← 必須按此   │
│  │         └─────────────────────────┘               │
│  │                    │                              │
│  │                    ▼                              │
│  │             換下一位玩家                           │
│  │                                                   │
│  └─ 選擇「猜牌」 → 猜牌流程 → 答案揭曉               │
│                                    │                 │
│                                    ▼                 │
│                          結算所有預測分數             │
└─────────────────────────────────────────────────────┘
```

**重要變更：回合不會自動結束，玩家必須手動按「結束」按鈕。**

### 2. 預測介面

問牌完成後，顯示預測選項與結束按鈕：

```
┌─────────────────────────────────────────┐
│           問牌完成！                      │
│                                         │
│   是否要預測蓋牌中有哪個顏色？            │
│                                         │
│   [紅色]  [黃色]  [綠色]  [藍色]         │
│    (點擊選擇，可不選)                    │
│                                         │
│   已選擇：紅色 ✓                         │
│                                         │
│           [結束回合]                     │
└─────────────────────────────────────────┘
```

**操作流程：**
1. 問牌完成後顯示此介面
2. 玩家可選擇一個顏色進行預測（可選）
3. 玩家按下「結束回合」
4. 換下一位玩家

### 3. 預測規則

| 項目 | 規則 |
|------|------|
| 誰可以預測 | 剛完成問牌的玩家 |
| 預測時機 | 問牌完成後、回合結束前 |
| 預測內容 | 從四個顏色中選擇**一個**顏色 |
| 預測次數 | 每次問牌後可預測一次 |
| 是否強制 | 非強制，玩家可選擇不預測 |
| **公開性** | **預測內容公開，所有玩家都能看到** |

### 4. 計分規則

當答案揭曉（有人猜牌成功或遊戲結束）時，結算所有預測：

| 預測結果 | 分數變化 |
|---------|---------|
| 預測的顏色**有在**蓋牌中 | **+1 分** |
| 預測的顏色**不在**蓋牌中 | **-1 分** |

**範例：**
- 蓋牌是 [紅色, 藍色]
- 小明預測「紅色」→ 紅色有在蓋牌中 → **+1 分**
- 小華預測「綠色」→ 綠色不在蓋牌中 → **-1 分**
- 小王選擇不預測 → **0 分**

### 5. 預測公開機制

預測動作是**公開**的，所有玩家都能看到：

```
┌─────────────────────────────────────────┐
│           遊戲紀錄                        │
│                                         │
│  小明 問牌後預測：[紅色]                  │
│  小華 問牌後預測：[綠色]                  │
│  小王 選擇不預測                          │
│                                         │
│  --- 答案揭曉 ---                        │
│  蓋牌為：[紅色] [藍色]                    │
│                                         │
│  預測結算：                              │
│  小明 預測 [紅色] ✓ +1分                 │
│  小華 預測 [綠色] ✗ -1分                 │
│                                         │
└─────────────────────────────────────────┘
```

### 6. 結果揭曉時機

預測結果在**答案翻開時**統一結算：
- 有人猜牌成功時
- 遊戲以其他方式結束時（如所有人出局）

## 技術實作

### 1. 遊戲狀態擴充

```javascript
// shared/constants.js 新增
const GAME_PHASES = {
  // ... 現有階段
  PREDICTING: 'predicting',  // 預測階段（問牌後）
};

// 遊戲狀態新增欄位
const gameState = {
  // ... 現有欄位
  predictions: [
    // 該局所有預測記錄
    {
      playerId: 'player1',
      color: 'red',        // 預測的顏色
      round: 3,            // 第幾回合預測的
      isCorrect: null,     // 答案揭曉後填入
    }
  ],
};
```

### 2. Socket 事件

```javascript
// 新增事件

// 前端 → 後端
'endTurn'              // 結束回合 { prediction: 'red' | null }
                       // prediction 為選擇的顏色，null 表示不預測

// 後端 → 前端
'postQuestionPhase'    // 進入問牌後階段（顯示預測選項）
'turnEnded'            // 廣播：玩家結束回合 { playerId, prediction }
'predictionsSettled'   // 答案揭曉，結算所有預測
```

### 3. 前端組件

```
frontend/src/components/
├── Prediction/
│   ├── index.js
│   ├── Prediction.js          // 預測主組件（選擇顏色）
│   ├── Prediction.css
│   ├── PredictionList.js      // 顯示所有人的預測
│   └── PredictionResult.js    // 預測結算結果
```

### 4. 流程說明

```
問牌完成後：
1. 顯示預測選項介面
2. 玩家可選擇一個顏色（或不選）
3. 玩家按下「結束回合」按鈕
4. 如果有選擇顏色 → 記錄預測
5. 廣播給所有玩家（顯示誰預測了什麼）
6. 換下一位玩家

答案揭曉時：
1. 取得所有該局的預測記錄
2. 比對每個預測與實際蓋牌
3. 預測對 +1 分，預測錯 -1 分
4. 廣播結算結果
```

# 本草 Herbalism 測試計畫書

**版本**：1.0
**建立日期**：2026-01-27
**專案版本**：1.0.160

---

## 一、測試概述

### 1.1 測試目的
確保「本草 Herbalism」桌遊網頁版的所有功能正確運作，包括遊戲邏輯、使用者介面、即時通訊、認證系統等。

### 1.2 測試範圍

| 測試類型 | 範圍 | 工具 |
|---------|------|------|
| 單元測試 | 工具函數、服務層、組件 | Jest, React Testing Library |
| 整合測試 | 前後端互動、Socket 通訊 | Jest, Socket.io-client |
| E2E 測試 | 完整使用者流程 | Playwright / Cypress |

### 1.3 測試環境

| 環境 | 說明 |
|------|------|
| 本地開發 | localhost:3000 (前端) + localhost:3001 (後端) |
| 測試伺服器 | Cloud Run 測試環境 |
| 瀏覽器 | Chrome, Safari, Firefox |
| 裝置 | Desktop, iPad, Mobile |

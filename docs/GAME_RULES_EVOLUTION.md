# 演化論：物種起源 遊戲規則

## 一、遊戲概述

《演化論：物種起源》(Evolution: The Origin of Species) 是一款 2-4 人的策略卡牌遊戲。玩家扮演造物主，創造各種生物並賦予牠們不同的性狀能力，在弱肉強食的世界中讓自己的生物存活下來。遊戲時間約 30-60 分鐘，適合 10 歲以上玩家。

## 二、遊戲配件

- 84 張雙面卡牌（一面為蜥蜴圖案代表生物，一面為性狀能力）
- 2 顆骰子
- 4 個計分棋子
- 1 張計分板
- 食物標記：
  - 15 個紅色標記（現有食物）
  - 10 個藍色標記（額外食物）
  - 10 個黃色標記（脂肪儲存）

## 三、遊戲設置

1. 將計分棋子放在計分板起始位置
2. 洗勻所有卡牌，正面朝下放置成牌庫
3. 每位玩家發 **6 張**手牌
4. 三種顏色的食物標記分類放在一旁備用
5. 擲骰決定起始玩家

## 四、回合流程

每回合分為四個階段：

### 第一階段：演化階段

從起始玩家開始，玩家輪流出牌。每次輪到時，玩家可以：

1. **創造生物**：將 1 張牌**蜥蜴面朝上**放在自己面前，代表創造了一隻新生物
   - 生物數量沒有上限

2. **賦予性狀**：將 1 張牌**蜥蜴面朝下**放在自己的生物下方，作為該生物的性狀
   - 如果卡牌有兩種性狀可選，必須選擇其中一種，選定後不能更改
   - 同一隻生物的性狀無上限，但不能有重複的性狀

3. **跳過 (Pass)**：不出牌

當所有玩家都選擇跳過時，進入下一階段。

### 第二階段：食物供給階段

起始玩家擲骰決定本回合的食物數量：

| 玩家人數 | 食物數量 |
|---------|---------|
| 2 人 | 1 顆骰子 + 2 |
| 3 人 | 2 顆骰子總和 |
| 4 人 | 2 顆骰子總和 + 2 |

從食物庫取出對應數量的**紅色食物標記**放在桌子中央。

### 第三階段：進食階段

從起始玩家開始，玩家輪流餵食自己的生物：

**基本規則：**
- 每隻生物的基本食量為 **1**
- 某些性狀會增加食量（卡牌左上角標示 +1 或 +2）
- 例如：有 +1 性狀的生物需要吃 2 個食物才算吃飽

**一般生物（非肉食）：**
- 從桌子中央取 1 個紅色食物
- 放到該生物上

**肉食生物：**
- **不能**吃紅色食物
- 必須攻擊其他生物（可以是自己的或其他玩家的）
- 在進食階段，每隻肉食生物只能攻擊一次
- 攻擊成功時：
  - 被攻擊的生物**滅絕**（生物卡和所有性狀卡棄置）
  - 攻擊者獲得 **2 個藍色額外食物**

**吃飽判定：**
- 當生物上的食物數量 = 該生物的食量需求時，視為吃飽
- 吃飽的生物不能再進食（除非有脂肪組織性狀）

**階段結束：**
- 當所有玩家都選擇跳過時，進入下一階段
- 桌子中央剩餘的食物**不保留**到下一回合

### 第四階段：滅絕與抽牌階段

**滅絕：**
- 所有**未吃飽**的生物滅絕
- 滅絕的生物連同其性狀卡一起放入該玩家的棄牌堆

**移除食物：**
- 移除所有生物上的紅色和藍色食物標記
- **黃色脂肪標記保留**在生物上

**抽牌：**
- 起始玩家發牌給每位玩家
- 每位玩家抽取 **存活的生物數量 + 2** 張牌
- 特殊情況：如果玩家沒有生物且沒有手牌，則補發 6 張牌

**回合結束：**
- 起始玩家標記傳給下一位玩家
- 開始新回合

## 五、性狀卡說明

### 肉食相關

| 性狀名稱 | 食量 | 效果 |
|---------|------|------|
| **肉食** | +1 | 不能吃現有食物，必須攻擊其他生物。攻擊成功獲得 2 個藍色食物，被攻擊者滅絕 |
| **腐食** | - | 當任何生物被肉食攻擊滅絕時，，從獵食成功的生物的擁有者順位開始檢查，所有玩家有腐食性狀的生物的其中一隻會獲得一個藍色食物。**不能與肉食同時擁有** |
| **銳目** | - |只有銳目動物可以獵食具有偽裝性狀的生物 |

### 防禦相關

| 性狀名稱 | 食量 | 效果 |
|---------|------|------|
| **偽裝** | - | 肉食生物必須擁有銳目性狀才能攻擊此生物 |
| **穴居** | - | 當此生物吃飽時，無法被攻擊 |
| **毒液** | - | 被攻擊滅絕時，攻擊者也會在滅絕與抽牌階段死亡 |
| **水生** | - | 只有同樣擁有水生的肉食生物才能攻擊此生物，有水生的肉食動物也無法攻擊沒有水生的動物 |
| **敏捷** | - | 每次被攻擊時，擲 1 顆骰子，骰出 4-6 則逃脫成功，攻擊無效，擲出1-3則逃脫失敗，對方獵食成功 |
| **巨化** | +1 | 只有同樣擁有巨化的肉食生物才能攻擊此生物，有巨化性狀的生物也可以獵食沒有巨化性狀 |
| **斷尾** | - | 被攻擊時，可棄置此生物的 1 張性狀卡來取消攻擊，剛剛獵食的生物可以獲得一個藍色食物 |
| **擬態** | - | 在整個進食階段可以使用一次，被攻擊時，可將攻擊轉移給**自己**另一隻**一定可以被獵食**的生物 |

### 進食相關

| 性狀名稱 | 食量 | 效果 |
|---------|------|------|
| **脂肪組織** | - | 吃飽後可繼續獲得食物，儲存為黃色脂肪標記（每張卡儲存 1 個）。可在進食階段消耗脂肪滿足食量 |
| **冬眠** | - | 可選擇跳過整個進食階段，視為吃飽。使用後此性狀卡橫置，等到下個進食階段結束才可再次使用。遊戲的最後一回合不能使用冬眠 |
| **寄生蟲** | +2 | 只能放在**對手**的生物上，增加該生物的食量需求 |
| **掠奪** | - | 可以偷取其他(對手或自己的生物)**未吃飽**生物身上的 1 個食物，一個階段限用一次 |

### 互動相關

**通用規則：**
- 互動性狀必須放在**同一玩家**的兩隻生物之間
- 同一對動物之間不能有重複的互動性狀
- 當其中一隻動物滅絕時，與牠相關的所有互動性狀卡也要一併棄掉

| 性狀名稱 | 效果 |
|---------|------|
| **溝通** | 當其中一隻生物**拿取**現有食物（紅）時，另一隻也從中央拿取 1 個紅色食物（如果中央還有食物）。**會連鎖觸發**。範例：A-B 溝通，B-C 溝通。當 A 拿取紅色食物 → B 也拿取紅色食物 → C 也拿取紅色食物（如果中央食物足夠）。 |
| **合作** | 當其中一隻生物**獲得**現有食物（紅）或額外食物（藍）時，另一隻獲得 1 個藍色額外食物。**會連鎖觸發**。範例：A-B 合作，B-C 合作。當 A 獲得紅色食物 → B 獲得藍色食物 → C 獲得藍色食物。 |
| **共生** | 放置時需指定其中一隻為「**代表動物**」，另一隻為「**被保護者**」。**進食限制**：代表動物吃飽前，被保護者不能獲得食物。**攻擊限制**：肉食生物只能攻擊代表動物，不能攻擊被保護者。範例：A-B 共生（A 為代表），B-C 共生（B 為代表）。進食時必須先餵飽 A，才能餵 B；餵飽 B 後才能餵 C。肉食只能攻擊 A ，不能直接攻擊B 或 C。 |

### 特殊能力

| 性狀名稱 | 食量 | 效果 |
|---------|------|------|
| **踐踏** | - |如果玩家擁有踐踏性狀的生物，可在進食階段輪到自己時，將桌面一個現有食物移除 |


## 六、計分方式

遊戲結束時，計算以下分數：

| 計分項目 | 分數 |
|---------|------|
| 每隻存活的生物 | **+2 分** |
| 存活生物身上的每張性狀卡 | **+1 分** |
| 性狀卡左上角標示 +1 | **額外 +1 分** |
| 性狀卡左上角標示 +2 | **額外 +2 分** |

**注意**：寄生蟲放在對手身上，但計分時算被放置者的分數。

**最高分者獲勝！**

平手時，比較棄牌堆的卡牌數量，**較多者**獲勝。

## 七、遊戲結束

當牌庫抽完後，進行**最後一個回合**。

最後一回合的滅絕階段結束後，遊戲結束，進入計分。

## 八、重要規則補充

1. **肉食不能有腐食**：同一隻生物不能同時擁有肉食和腐食性狀
2. **燃燒脂肪不算獲得食物**：消耗脂肪標記不會觸發溝通、合作等性狀效果
3. **互動性狀**：溝通、合作、共生必須放在兩隻生物之間才有效
4. **抽牌不足**：如果發牌過程中牌庫用盡，某些玩家可能拿不到牌

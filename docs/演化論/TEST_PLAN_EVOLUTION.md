## 1. 測試概述

### 1.1 測試目的

本測試計畫旨在全面驗證「演化論：物種起源」遊戲的所有功能模組，確保：
- 遊戲邏輯符合規則書定義
- 前後端整合運作正常
- 多人即時對戰功能穩定
- 使用者體驗流暢無阻

### 1.2 測試範圍

| 模組 | 描述 | 優先級 |
|------|------|--------|
| 房間管理 | 創建、加入、離開房間 | P0 |
| 遊戲流程 | 四階段回合制 | P0 |
| 卡牌系統 | 84張雙面卡、19種性狀 | P0 |
| 生物系統 | 創建、性狀、進食、滅絕 | P0 |
| 戰鬥系統 | 肉食攻擊、防禦機制 | P1 |
| 互動性狀 | 溝通、合作、共生 | P1 |
| 計分系統 | 遊戲結束分數計算 | P1 |
| Socket 通訊 | 即時同步 | P0 |

### 1.3 測試類型

```
┌─────────────────────────────────────────────────────────────┐
│                      E2E 測試 (端對端)                        │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              整合測試 (Integration)                    │   │
│  │  ┌─────────────────────────────────────────────┐   │   │
│  │  │            單元測試 (Unit)                    │   │   │
│  │  │  cardLogic | creatureLogic | feedingLogic  │   │   │
│  │  │  phaseLogic | gameLogic | evolutionStore   │   │   │
│  │  └─────────────────────────────────────────────┘   │   │
│  │  Socket.io + Room Manager + Game Logic             │   │
│  └─────────────────────────────────────────────────────┘   │
│  完整遊戲流程：房間 → 演化 → 進食 → 滅絕 → 計分            │
└─────────────────────────────────────────────────────────────┘
```

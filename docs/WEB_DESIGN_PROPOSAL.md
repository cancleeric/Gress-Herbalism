# 本草 Herbalism - 網頁設計計畫書

## 一、專案概述

### 1.1 專案名稱
**本草 Herbalism** - 多人線上推理桌遊

### 1.2 專案簡介
這是一款 3-4 人的線上推理卡牌遊戲。玩家透過「問牌」收集資訊，推理出桌面上兩張蓋牌的顏色，最先達到 7 分的玩家獲勝。

### 1.3 目標用戶
- 喜歡推理遊戲的玩家
- 想與朋友線上同樂的用戶
- 年齡層：15-35 歲

### 1.4 線上版本
- **前端**：https://herbalism-frontend-130514813450.asia-east1.run.app
- **後端**：https://herbalism-backend-130514813450.asia-east1.run.app
- **GitHub**：https://github.com/nicholaswang941013/Gress-Herbalism

## 三、頁面規劃

### 3.1 頁面清單

| 頁面 | 路由 | 說明 |
|------|------|------|
| 登入頁 | `/` | 用戶登入/註冊 |
| 大廳 | `/lobby` | 房間列表、創建房間 |
| 遊戲房間 | `/game/:id` | 主要遊戲畫面 |
| 個人檔案 | `/profile` | 玩家統計資料 |
| 排行榜 | `/leaderboard` | 玩家排名 |
| 好友 | `/friends` | 好友管理 |

### 3.2 頁面詳細設計

#### 登入頁 (`/`)
- Logo 展示
- Google 登入按鈕
- 訪客登入選項
- 遊戲簡介

#### 大廳 (`/lobby`)
- 用戶資訊顯示（頭像、暱稱、分數）
- 房間列表（房間名、人數、狀態）
- 創建房間按鈕
- 快速加入按鈕
- 導航選單（排行榜、好友、個人檔案）

#### 遊戲房間 (`/game/:id`)
**等待階段：**
- 玩家列表（3-4 人）
- 準備狀態顯示
- 房主開始遊戲按鈕
- 離開房間按鈕

**遊戲階段：**
- 蓋牌區（2 張蓋牌）
- 玩家手牌區
- 其他玩家資訊（名稱、手牌數、分數）
- 行動按鈕（問牌/猜牌）
- 遊戲紀錄面板
- 當前回合提示

**結算階段：**
- 結果顯示（猜對/猜錯）
- 分數變化
- 下一局/返回大廳按鈕

## 四、遊戲核心元素

### 4.1 卡牌設計

**牌組配置（共 14 張）：**

| 顏色 | 數量 | 建議配色 |
|------|------|----------|
| 紅色 | 2 張 | #E53935 |
| 黃色 | 3 張 | #FDD835 |
| 綠色 | 4 張 | #43A047 |
| 藍色 | 5 張 | #1E88E5 |

### 4.2 UI 組件需求

#### 卡牌組件
- 正面：顯示顏色
- 背面：統一圖案（蓋牌用）
- 選中狀態：邊框高亮
- 禁用狀態：灰色遮罩

#### 顏色選擇器
- 雙色組合卡片（6 種組合）
- 紅黃、紅綠、紅藍、黃綠、黃藍、綠藍

#### 玩家資訊卡
- 頭像
- 暱稱
- 手牌數量
- 當前分數
- 活躍/退出狀態

#### 遊戲紀錄
- 時間軸形式
- 顯示：問牌、猜牌、預測等動作
- 可滾動瀏覽

## 五、遊戲流程

### 5.1 基本流程

```
遊戲開始
    │
    ▼
┌─────────────────┐
│  發牌（每人3-4張）│
│  抽蓋牌（2張）    │
└─────────────────┘
    │
    ▼
┌─────────────────┐
│  玩家回合        │◄──────┐
│  選擇：問牌/猜牌  │       │
└─────────────────┘       │
    │                     │
    ├── 問牌 ──┐          │
    │         ▼          │
    │    選擇兩色         │
    │    選擇目標         │
    │    選擇方式         │
    │    （可選）預測      │
    │    結束回合 ────────┘
    │
    └── 猜牌 ──┐
              ▼
         選擇兩色猜測
              │
              ▼
         跟猜階段
              │
              ▼
         結算結果
              │
    ┌─────┴─────┐
    ▼           ▼
  猜對        猜錯
  +3分        退出當局
    │           │
    ▼           ▼
  達7分？    還有人？
    │           │
   是/否      是/否
    │           │
    ▼           ▼
 遊戲結束    繼續/下一局
```

### 5.2 計分規則

| 動作 | 分數 |
|------|------|
| 猜牌正確 | +3 分 |
| 跟猜正確 | +1 分 |
| 跟猜錯誤 | -1 分 |
| 預測正確 | +1 分 |
| 預測錯誤 | -1 分 |
| 勝利門檻 | 7 分 |

## 六、設計風格建議

### 6.1 整體風格
- **主題**：中國風草藥/自然風格
- **色調**：暖色系為主，綠色點綴
- **字體**：清晰易讀，支援中文

### 6.2 配色建議

| 用途 | 顏色 | 色碼 |
|------|------|------|
| 主色 | 深綠 | #2E7D32 |
| 輔色 | 米白 | #FFF8E1 |
| 強調 | 金色 | #FFB300 |
| 背景 | 淺灰 | #F5F5F5 |
| 文字 | 深灰 | #424242 |

### 6.3 動畫效果
- 卡牌翻轉動畫
- 發牌動畫
- 得分動畫（+3、-1 等）
- 按鈕互動效果
